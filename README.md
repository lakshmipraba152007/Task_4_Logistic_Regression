 # 🩺 Breast Cancer Classification Using Logistic Regression

## 📌 Project Overview

This project demonstrates how to build a **Binary Classification** model using **Logistic Regression** with the **Breast Cancer Wisconsin Dataset**. The workflow covers data preprocessing, feature scaling, model training, prediction, and performance evaluation using industry-standard classification metrics.

---

## 🎯 Objective

Develop a Logistic Regression model to accurately classify breast cancer tumors as **Malignant** or **Benign** while understanding the concepts of binary classification, probability prediction, and model evaluation.

---

## 🚀 Features

* 📥 Load the Breast Cancer Wisconsin Dataset
* 🧹 Data preprocessing and feature scaling
* ✂️ Train-test data splitting
* 🤖 Train a Logistic Regression model
* 📊 Predict tumor classes
* ✅ Evaluate model performance
* 📈 Visualize the Confusion Matrix
* 📉 Plot the ROC Curve
* 🎯 Calculate ROC-AUC Score
* ⚙️ Tune the decision threshold
* 📐 Visualize the Sigmoid Function
